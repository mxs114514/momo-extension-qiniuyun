  目标行为
  插件开始工作后，页面显示右下角悬浮球。点击悬浮球后，在页面底部居中展开长方形字幕面板。面板右上角有 -，点击后
  收回成悬浮球。

  关键约束
  展开/收起只是 UI 展示状态，不影响翻译状态：

- 不暂停翻译
- 不继续翻译
- 不停止翻译
- 不清空字幕
- 不改变当前 controller 状态

  状态表现

- 翻译中：面板显示最新中文字幕，收起后后台继续更新。
- 已暂停：保留最后一句字幕，仍可展开/收起。
- 停止：清空字幕面板内容，但保留悬浮球入口。
- 异常：面板可显示异常提示，也可收起。

      - 收到翻译快照后默认显示悬浮球。
      - 点击悬浮球后显示底部居中字幕面板。
      - 点击 - 后回到悬浮球。
      - 展开/收起不发送任何翻译控制消息。
      - 暂停状态下仍保留字幕和切换能力。

  2. 再改 content-script.ts：
      - 从“有字幕才创建字幕层”改成“运行态创建浮层容器”。

      - UI 切换只改 DOM，不碰翻译链路。
