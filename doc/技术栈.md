# AI 同声传译助手技术栈

## 当前一期

| 类别 | 选型 |
| --- | --- |
| 前端框架 | React 19 |
| 开发语言 | TypeScript 6 |
| 构建工具 | Vite 8 |
| 包管理 | pnpm |
| 样式 | Tailwind CSS 4 |
| 音频采集 | MediaDevices |
| 音频处理 | Web Audio API、AudioWorklet |
| 实时通信 | WebSocket |
| 签名 | Web Crypto API |
| 语音翻译 | 腾讯云实时语音翻译 |
| 单元测试 | Vitest |
| 组件测试 | Testing Library |
| 端到端测试 | Playwright |

## 浏览器插件阶段

| 类别 | 选型 |
| --- | --- |
| 扩展框架 | WXT |
| 浏览器规范 | Manifest V3 |
| 标签页音频 | `chrome.tabCapture` |
| 持续音频任务 | Offscreen Document |
| 页面字幕 | Content Script |
| 跨上下文通信 | Chrome Messaging |

## 后续阶段

| 阶段 | 增量技术 |
| --- | --- |
| 二期：历史记录与导出 | IndexedDB、Dexie、Blob API |
| 三期：AI 翻译官 | 角色组件、可选 Zustand、可选 LLM |
| 四期：Live2D 翻译官 | Live2D SDK |

## 按需引入

- shadcn/ui：设置页、历史记录页或复杂交互出现后引入。
- Zustand：多页面 UI 状态明显复杂后引入。
- Zod：复杂配置、数据迁移或导入导出出现后引入。

具体架构和实施指导见 `技术实施指南.md`。
